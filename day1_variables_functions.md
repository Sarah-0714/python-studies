DAY 1 - VARIABLES AND FUNCTIONS

Today i learned the basics of variables and functions in python. A variable is used to store information that can be used later in a program. It can hold different types of data like
numbers and text.
EXAMPLE:
name="Sarah"
age="19"
I also learned about functions. Functions help us group a set of instructions together so that we can use them whenever needed without writing the same code again.
EXAMPLE: 
def greet():
print("Hello")
greet()
After learning this topic, i understood that variables make it easy to manage data, while functions helpkeep the code simple, organized and resuable
